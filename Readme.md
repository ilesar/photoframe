### Setup

Copy the file `.env.example` to `.env`

Get the ID by creating a shareable album URL and copy the hashed string at the end of the route. Set the ID inside the .env file.


### Install

```
npn install
```

### Start

```
node index.js
```

### Use

Visit http://localhost:8123
